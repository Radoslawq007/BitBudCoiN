# BitBudCoiN

BitBudCoiN to lekki, edukacyjny projekt blockchain stworzony w celu pokazania, jak działa:
- sieć peer‑to‑peer,
- kopanie bloków,
- portfel,
- explorer,
- backend API,
- frontend hostowany na GitHub Pages.

Projekt składa się z dwóch części:

## 🔧 Backend (Node.js)
Backend odpowiada za:
- tworzenie bloków,
- obsługę transakcji,
- API dla frontendu,
- prostą logikę blockchain.

Backend uruchamiasz lokalnie lub na serwerze (Railway / Render / VPS).

## 🎨 Frontend (statyczny)
Frontend zawiera:
- `index.html` — strona główna
- `wallet.html` — portfel
- `explorer.html` — explorer bloków
- `miner.html` — kopanie bloków
- `network.html` — status sieci

Frontend działa na GitHub Pages
